== Description ==

### Alobaidi Slider 

Beautiful slider, responsive and retina, autoplay, touch devices, youtube, vimeo, keek, and instagram image support, slider widget, easy to use, compatible with all major browsers.

### The Features

1. Fully Responsive.
2. Retina Ready.
3. Unlimited amount of images and videos to be added to the slider. 
4. Unlimited Sliders.
5. Autoplay Support (will be stopped when hover).
6. Slider Widget.
7. Custom wrap width and wrap height.
8. YouTube and Vimeo and Keek support.
9. Instagram images support.
10. Images Captions.
11. Touch devices support.
12. Compatible with any lightbox.
13. Compatible with all major browsers, like Google Chrome, FireFox, Opera, Safari, IE9, IE10, IE11, iOS browser, Android browser, etc.
14. Easy to use and without options, 2 shortcodes only.

== Installation ==

### Installation

1. Upload 'alobaidi-slider' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Now go to Plugins menu > Alobaidi Slider.
4. For widget: go to Appearance menu > Widgets > Alobaidi Slider.
5. [Read explanation of use.](http://wp-plugins.in/alobaidi-slider)
